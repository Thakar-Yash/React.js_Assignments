React-Redux Project
This repository contains three React applications that demonstrate the use of Redux for state management:

To-Do List – A task management app where users can add, delete, and mark tasks as completed.
Counter – A simple counter app with increment, decrement, and reset functionality.
CRUD Operations – A basic CRUD (Create, Read, Update, Delete) application to manage a list of items.
Technologies Used

Redux
Redux Toolkit (if used)
